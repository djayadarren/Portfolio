Projects

Self-Balancing Robot

A project that brought together control theory, embedded systems, and (a lot of) fine-tuning. This is our two-wheeled robot that balances itself in real time using a PID controller and IMU data. We tuned the control loop through hands-on testing to get it stable, responsive, and surprisingly agile. The robot also includes edge detection with a time-of-flight sensor, object detection using a compact camera, and Bluetooth connectivity for wireless control and feedback.
