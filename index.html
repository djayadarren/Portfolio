Projects

Self-Balancing Robot

A project that brought together control theory, embedded systems, and (a lot of) fine-tuning. This is a two-wheeled robot that balances itself in real time using a PID controller and IMU data. The control loop is tuned through hands-on testing to get it stable, responsive, and surprisingly agile. The robot also includes edge detection with a time-of-flight sensor, object detection using a compact camera, and Bluetooth connectivity for wireless control and feedback.



Snake Game

A simple snake game built in Python using Tkinter. It features a custom GUI with animated menus, real-time canvas rendering, and smooth input handling via after() callbacks. Players can select snake colors and difficulty before starting the game, which includes dynamic effects like wavy snake motion and pulsing food. The game is organized as a little object-oriented programming project, using class-based structure for clear separation of logic, UI, and animation timing.



Metal Detecting Robot
